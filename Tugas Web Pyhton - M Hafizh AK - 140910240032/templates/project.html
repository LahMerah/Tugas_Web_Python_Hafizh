Project

Project yang pernah saya lakukan sampai sekarang baru hanya satu, yaitu membuat alat penyiram tanaman otomatis menggunakan Arduino. Project ini dilakukan sembari memenuhi tugas akhir dari mata kuliah pemrograman. Alasan saya memilih project ini adalah karena saya sering sekali melihat orang tua saya menyiram tanaman di pagi dan sore hari sehingga saya terfikir untuk membuat alat yang dapat membantu menyiram tanaman terutama ketika tidak ada orang yang berada dirumah. Oleh karena itu, saya memutuskan untuk memilih project ini.

Alat dan bahan yang digunakan untuk membuat project ini adalah :

  1. Arduino UNO
  2. LCD
  3. Sensor Soil Moisture
  4. Relay
  5. Kabel USB
  6. Kabel Jumper
  7. Pompa Air
  8. Baterai
  9. Obeng
  10. Gunting

Untuk cara kerja dari alat ini adalah :

  1. Pertama, alat ini akan membaca kelembapan tanah menggunakan sensor soil moisture.
  2. Jika kelembapan tanah kurang dari 30%, maka pompa air akan menyala dan LCD akan menampilkan tulisan "Tanah Kering"
  3. Jika kelembapan tanah lebih dari 30%, maka pompa air akan mati dan LCD akan menampilkan tulisan "Tanah Basah"
  4. Relay pada alat ini berguna sebagai switch yang menyala/menonaktifkan pompa air yang terhubung ke selang

ini memakan waktu sekitar 1 minggu diamana saya tidak bekerja sendirian tapi ada kelompok yang ikut membantu project ini. bagian tersulit dari project ini adalah saya dan teman kelompok saya baru pertama kali merancang menggunakan arduino uno sehingga kami lumayan kebingungan cara kerjanya. dibagian pengcodean juga kami kesusahan karena lumayan membingungkan cara kerja code dengan arduino UNO.
